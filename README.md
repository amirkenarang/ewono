## Description

Ewono is a framework for building server-side applications in Node.js. It is built on top of [NestJS](https://nestjs.com/) and provides a set of high-level abstractions for common tasks and save english words with the meaning and usage of the word.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
